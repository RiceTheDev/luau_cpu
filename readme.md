## Specifications

### CPU
- **8-bit running assembly-like instructions (math, jumps, draw and more)**
- **16 registers**

### GPU
- **16x12 monochrome screen**
- **128 bytes of VRAM**
- **Text display for printing and debugging**

### RAM
- **256 bytes of RAM**
- **RAM is too expensive, we couldn't get more 😭**
